Production portal managing paramedical professional registrations and certificate issuance for the state government.
- Full **Laravel + Blade** stack with **SMS OTP-based login** for secure authentication
- **Role-Based Access Control (RBAC)** for multi-level government user management
- End-to-end registration workflows, document verification, and PDF certificate generation (DomPDF)
- Deployed on **AWS EC2 & S3**

---

### 🏟️ Sports Venue Booking Management System — JSSPS
> Jharkhand State Sports Promotion Society | Govt. of Jharkhand | Laravel + Next.js 14 + TypeScript | AWS

Full-featured booking and accommodation management system for state sports facilities.
- **Laravel REST API + Next.js 14 + TypeScript** frontend with **JWT authentication** and **RBAC**
- Real-time booking status, room allotment with race-condition protection (`lockForUpdate()`), payment scrutiny, and analytics dashboards (Recharts)
- Multi-role workflow: applicants → scrutiny officers → HOD approvals
- Deployed on **AWS EC2 & S3**
